Web programming also known as web development is the creation of dynamic web applications. Some of the famous examples of web applications are social networking sites such as Gaana or JioSavn sites like Spotify . There are two broad divisions of web development – front-end development also called client-side development and back-end development also called server-side development. Front-end development refers to constructing what a user sees when they load a web application – the content, design and how you interact with it. Back-end development controls what goes on behind the scenes of a web application. A backend often uses a database to generate the front-end. The traditional way of learning web development is by enrolling into the course of web development. There are a plenty of websites that offers free as well as paid courses of web development. One can also learn web development by inspecting the source code of a web page which shows all the code that went into creating the web page. Web developers often work for clients who are trying to get their product or service onto the web. The work is typically very project focused and involves collaborating with a team that helps to coordinate the client’s needs into the product. The client could be a tech company, an organization, or a government. The work could involve front-end, back-end, or full-stack web development. In short web developers build and maintain websites.
